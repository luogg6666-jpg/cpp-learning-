## 思考了很久指针知识

# &就是取地址，*就是根据地址找值
# int* p = &a; 的意思其实就是 “定义一个叫p的变量专门用来存int的地址”，这个p被称为指针变量，是专门用来存地址的，指向的是a所在的地址
# *p就是 “根据p存的地址来找值”，比如这里就是p存着a的地址，那么*p就是去a所在的地址去找值
# 引用是指针的一个语法糖，其实就是给一个变量起一个别名，两者本质上还是同一个变量，两者在汇编语言上的表示上其实是一模一样的
# 在写函数时，想带走值其实有几个方法
# 最优先的方法:使用return,在数据占用内存小时极为好用。
### 示例代码
```cpp
#include <iostream>

#define LOG(x) std::cout<< x << std::endl

int Increment(int value)
{
	value++;
	return value;
}

int main()
{
	int a = 5;
	a = Increment(a);

	LOG(a);

	std::cin.get();
}
```
### 踩的坑
# 	a = Increment(a);  忘记接return的返回值，没写前面的a = ,导致a没变化
然后才是用引用，这个相当于直接去变量“家”改变它的数值，一般用于处理数据占用内存较大时的情况，而且这种方法可以避免用指针那一套，简化了过程。
最后才是选用指针那一套
